# Levels of Processing: A Framework for Memory Research

**Author(s):** Fergus I. M. Craik (University of Toronto), Robert S. Lockhart (University of Toronto)
**Published:** *Journal of Verbal Learning and Verbal Behavior*, 11(6), 671-684, 1972
**DOI/Link:** https://doi.org/10.1016/S0022-5371(72)80001-X
**Field:** Cognitive Psychology / Memory

---

## Abstract

Craik and Lockhart propose the levels-of-processing framework as an alternative to the prevailing multi-store model of memory (Atkinson & Shiffrin, 1968). Rather than conceptualizing memory as a sequence of stores (sensory, short-term, long-term) through which information passes, they propose that memory durability is determined by the depth of processing at the time of encoding. Shallow processing -- attending to the physical or surface features of a stimulus (its font, its sound) -- produces fragile, rapidly decaying memory traces. Deep processing -- attending to the meaning, the associations, the semantic content -- produces durable, long-lasting traces. The framework shifts the focus of memory research from storage structures to encoding processes.

## Key Findings

- **Depth of processing determines retention.** In the canonical paradigm, participants are given incidental learning tasks that require different levels of processing: (a) structural -- "Is this word in uppercase?" (shallow); (b) phonological -- "Does this word rhyme with X?" (intermediate); (c) semantic -- "Does this word fit in the sentence 'The ___ was on the table'?" (deep). On a surprise memory test, semantic processing produces dramatically better recognition and recall than structural or phonological processing, even though participants did not intend to learn.

- **The critical variable is processing, not intention.** Whether participants intend to learn is less important than how they process the information. Incidental learning with deep processing produces better retention than intentional learning with shallow processing. This finding overturns the intuition that wanting to remember is what makes memory work -- it is the nature of the cognitive engagement, not the intention, that determines durability.

- **Type I vs. Type II rehearsal.** The framework distinguishes two types of rehearsal: maintenance rehearsal (Type I) -- repeating information to keep it active in working memory without processing it further -- and elaborative rehearsal (Type II) -- processing information more deeply by connecting it to existing knowledge, forming images, generating associations. Only Type II rehearsal improves long-term retention. Simply repeating information does not deepen its processing.

- **The framework is about a continuum, not discrete stages.** Although often described as "three levels," Craik and Lockhart proposed a continuum from shallow to deep processing. The structural/phonological/semantic distinction is an experimental operationalization, not a theoretical claim about three discrete stages. In practice, processing depth is continuous.

## Methodology

This is a theoretical framework paper, not an experimental report. Craik and Lockhart synthesize existing evidence from: (a) incidental learning experiments (Hyde & Jenkins, 1969, 1973) showing that semantic orienting tasks produce better memory than structural or phonological tasks; (b) studies of rehearsal showing that maintenance rehearsal does not improve long-term retention; (c) evidence that "deeper" processing takes more time and produces better memory across multiple paradigms. The paper proposes the framework and identifies the evidence that supports it, rather than reporting new experiments.

## Relevance to Learn Mode

**Principle:** #2 -- Content Before Learner; #5 -- Push Up the Stack
**Specific connection:** The levels-of-processing framework is the theoretical backbone of the encoding stack. The stack's six levels (Receive through Teach) map directly onto the depth-of-processing continuum: Receive engages only shallow processing (reading without generating); Retrieve forces search through semantic memory; Apply requires semantic deployment in context; Simulate requires semantic construction; Connect requires cross-domain semantic elaboration; Teach requires metacognitive semantic organization. Each level demands deeper processing than the one below it. The specification's insistence that every interaction reach at least Apply (Layer 3) is a direct implementation of Craik and Lockhart's finding: shallow processing produces fragile traces, and only deeper processing produces durable learning.
**What this paper provides that others don't:** The theoretical framework that unifies the encoding stack. Roediger and Karpicke demonstrate the testing effect. Slamecka and Graf demonstrate the generation effect. Craik and Lockhart provide the explanation for WHY both work: they force deeper processing. Without the levels-of-processing framework, the testing effect and generation effect are isolated findings. With it, they are instances of a single principle -- deeper processing produces more durable traces.

## Key Quotes

> "Trace persistence is a function of depth of analysis, with deeper levels of analysis associated with more elaborate, longer lasting, and stronger traces."

> "The memory trace can be thought of as a by-product of perceptual analysis, and its durability is a positive function of the depth to which the stimulus has been analyzed."

> "It is the nature of the encoding operations, not the intention to learn, that determines the persistence of the memory trace."

## Limitations

The framework has been criticized for circularity: deep processing is defined as processing that produces good memory, and good memory is attributed to deep processing. Craik and Lockhart acknowledged this concern and proposed that depth should be defined independently of retention (e.g., by processing time, brain activation, or the nature of the orienting task). Subsequent research has largely resolved the circularity by identifying independent markers of depth (fMRI studies showing left prefrontal activation for semantic processing). The framework also does not account well for the effects of retrieval practice -- re-reading a word (shallow) produces less learning than recalling it (deep), but the levels-of-processing framework does not naturally distinguish between encoding and retrieval operations. The retrieval science program (Bjork, Roediger & Karpicke) fills this gap.
